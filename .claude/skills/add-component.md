# add-component

Создаёт переиспользуемый компонент в `src/components/`.

## Что делает
1. Создаёт файл `src/components/<ComponentName>.tsx`, функциональный компонент,
   экспорт `default`.
2. Описывает пропсы через `interface <ComponentName>Props`.
3. Стилизует через Tailwind-классы (без инлайн-стилей), классы группирует:
   макет → типографика → цвета → состояния.

## Правила
- Никаких `any`: типизировать пропсы явно.
- Условные классы — через шаблонные строки, без `cn`/`classnames`.
- Иконки — эмодзи.
- Общие типы (из данных) выносить в `src/types.ts`, локальные пропсы держать рядом.

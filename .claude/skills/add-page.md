# add-page

Создаёт новую страницу приложения.

## Что делает
1. Создаёт файл `src/pages/<PageName>.tsx` (имя с большой буквы), функциональный
   компонент, экспорт `default`.
2. Добавляет маршрут в `src/App.tsx` (`<Route path=... element={<PageName />} />`).
3. Подключает загрузку данных через `useSWR` с `fetcher` из `src/lib/api.ts`.
4. Обрабатывает три состояния: loading (скелет/«Загрузка…»), error (сообщение об
   ошибке), empty (пустой список).
5. Использует общие классы `card` / `btn-primary` и Tailwind для вёрстки.

## Правила
- Типы данных страницы — в `src/types.ts`.
- URL-хелперы для запросов — в `src/lib/api.ts`, не хардкодить строки в компоненте.
- Иконки — эмодзи.
